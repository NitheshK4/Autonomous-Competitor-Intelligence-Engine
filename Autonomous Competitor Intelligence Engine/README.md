# Autonomous Competitor Intelligence Engine

This folder contains stronger, clearer prompts for your internship project.

Use the files in this order:

1. `01_MASTER_BUILD_PROMPT.md`
   Use this first. It is the main full-project build prompt.
2. `02_UI_AND_EXTENSION_POLISH_PROMPT.md`
   Use this after the first build if you want Antigravity to improve the UI, onboarding flow, and Chrome extension quality.
3. `03_FINAL_AUDIT_PROMPT.md`
   Use this last. It tells Antigravity to do a full no-skip audit against every assignment requirement.
4. `REQUIREMENTS_CHECKLIST.md`
   Keep this open while reviewing output so nothing gets missed.

Project name to use everywhere:

- `Autonomous Competitor Intelligence Engine`

Recommended flow:

1. Paste `01_MASTER_BUILD_PROMPT.md` into Antigravity.
2. Let it generate the project.
3. If needed, paste `02_UI_AND_EXTENSION_POLISH_PROMPT.md`.
4. Before submission, paste `03_FINAL_AUDIT_PROMPT.md`.
5. Compare the result against `REQUIREMENTS_CHECKLIST.md`.
